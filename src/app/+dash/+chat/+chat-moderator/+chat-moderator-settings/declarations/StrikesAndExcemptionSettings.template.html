Strikes and Group Excemption

Messages by users in the excempted group (or higher) are not evaluated by the chat moderator.
For example if you set the excempted group to Viewer all users will be excempted from moderation.

Biliomi keeps moderation strikes per user for the duration of an hour.
The first moderation action will perform strike 1, the second Strike 2 and 3 and beyond will execute Strike 3.
